# Golf Stats Support

Support and privacy information for **Golf Stats**, an iPhone/iPad app for
tracking golf tournament statistics, round by round and hole by hole.

## Support

Need help with the app, found a bug, or have a feature request? Open a GitHub
issue and I'll respond there:

**<https://github.com/terumorimura/golf-stats-support/issues>**

When reporting a problem, please include your device model, iOS version, and
the steps to reproduce the issue.
